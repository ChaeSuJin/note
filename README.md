## 랜덤(Random) #다음 띄어쓰기

랜덤은 뭐가 나올지 예측이 불가능한 값을 말한다  
랜덤을 만드는 방버을 여러 가지가 있습니다.(줄바꿈 스페이스 두번)

1. Random이라는 도구를 생성하여 사용 (1. 찍고 띄어씀)
2. Math.random() 명령을 사용
3. secureRandom 도구를 생성하여 사용

여기서는 Random 도구를 생성한다

(백킹 띄어쓰기 안하고 언어 - 코드블럭 쓰는 법)
```java 
Random r = new Random();
```
